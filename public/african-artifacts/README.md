# African Artifact Background Assets

Place your licensed images here to power the client dashboard background.

## Expected Filenames

- `african-artifact-1.jpg`
- `african-artifact-2.jpg`
- `african-artifact-3.jpg`

## Recommended Specs

- Format: JPG or WebP
- Resolution: 1600×1600 or higher
- Subject ideas: African artifacts, textiles, carved patterns, pottery, beadwork
- Keep subjects centered with some breathing space so they can be cropped safely

## Notes

- The dashboard will automatically use these images if they exist.
- If any image is missing, the background falls back to the patterned gradient.
